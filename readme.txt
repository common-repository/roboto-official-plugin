== Description ==

Personalize your WordPress blog with your face! Robo.to lets you record 4-second video avatars as status updates you can share with whoever you like.

Plugin Features

1. Add your latest robo.to update using the shortcode [robo.to]
2. Add your latest robo.to update to your theme's sidebar

Don't have one yet? Get yours here: http://www.robo.to

== Installation ==


1. Upload `roboto.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your information in the settings -> robo.to 

Add your latest Robo.to update to any post or page with the shortcode [robo.to].

Drag the Robo.to widget to your sidebar in the Appearance -> Widgets panel.
